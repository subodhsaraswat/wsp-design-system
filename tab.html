Tabs

Tabs are used to separate content into different panes where each pane is viewable one at a time.
Just add base class .wsp-nav or .wsp-nav-tabs to any ul , then extend with custom styles or our various included modifier classes.

Example

  • Tab1
  • Tab2
  • Tab3
  • Tab4
<ul class="wsp-nav wsp-nav-tabs">
    <li class="wsp-nav-item">
       <a class="wsp-nav-link active" href="#">Tab1</a>
    <li>
    <li class="wsp-nav-item">
       <a class="wsp-nav-link" href="#">Tab2</a>
    <li>
    <li class="wsp-nav-item">
       <a class="wsp-nav-link" href="#">Tab3</a>
    <li>
    <li class="wsp-nav-item">
       <a class="wsp-nav-link disabled" href="#">Tab4</a>
    <li>
<ul>
