Buttons

Use button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Example

<button type="button" class="wsp-button button-primary">Primary</button>
<button type="button" class="wsp-button button-secondary">Secondary</button>
<button type="button" class="wsp-button button-success">Success</button>
<button type="button" class="wsp-button button-danger">Danger</button>
<button type="button" class="wsp-button button-warning">Warning</button>
<button type="button" class="wsp-button button-info">Info</button>
<button type="button" class="wsp-button button-link">Link</button>

Sizes

Fancy larger or smaller buttons? Add .button-lg or .button-sm for additional sizes.

<button type="button" class="wsp-button button-primary button-lg">Large button</button>
<button type="button" class="wsp-button button-secondary button-lg">Large button</button>
<button type="button" class="wsp-button button-primary button-sm">Small button</button>
<button type="button" class="wsp-button button-primary button-sm">Small button</button>

Button block

Create block level buttons—those that span the full width of a parent—by adding .button-block.

<button type="button" class="wsp-button button-primary button-block">Block level</button>
<button type="button" class="wsp-button button-secondary button-block">Block level</button>

Button Outline

In need of a button, but not the hefty background colors they bring? Replace the default modifier classes with the .button-outline-* ones to remove all background images and colors on any button.

<button type="button" class="wsp-button button-outline-primary">Primary</button>
<button type="button" class="wsp-button button-outline-secondary">Secondary</button>
<button type="button" class="wsp-button button-outline-success">Success</button>
<button type="button" class="wsp-button button-outline-danger">Danger</button>
<button type="button" class="wsp-button button-outline-warning">Warning</button>
<button type="button" class="wsp-button button-outline-info">Info</button>

Active State

Buttons will appear pressed (with a darker background, darker border, and inset shadow) when active.

<button type="button" class="wsp-button button-primary active">Active Button</button>
<button type="button" class="wsp-button button-secondary active">Active Button</button>

Disable State

Make buttons look inactive by adding the disabled boolean attribute to any <button> element.

<button type="button" class="wsp-button button-primary" disabled">Active Button</button>
<button type="button" class="wsp-button button-secondary" disabled>Active Button</button>
