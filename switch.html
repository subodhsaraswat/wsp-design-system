Switch

Use the switch option, instead of a checkbox, for simple “On/Off” options.

Default Switch

Copy <label class="wsp-switch">
    <input type="checkbox">
    <span class="slider"></span>
</label>

Active Switch

For active pass the checked property.

Copy <label class="wsp-switch">
    <input type="checkbox" checked>
    <span class="slider"></span>
</label>

Disable Switch

For active pass the disabled property.

Copy <label class="wsp-switch">
    <input type="checkbox" disabled>
    <span class="slider"></span>
</label>
