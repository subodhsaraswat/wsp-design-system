Input Group

Using the contextual .wsp-form-* classes on an <input> element quickly provide contextual input.

Form controls

Textual form controls—like <input>s, <select>s, and <textarea>s —are styled with the .wsp-form-control class. Included are styles for general appearance, focus state, sizing, and more.

Copy <form>
  <div class="wsp-form-group">
    <label for="exampleFormControlInput1">Email address</label>
    <input type="email" class="wsp-form-control" id="exampleFormControlInput1" placeholder="name@example.com">
  </div>

  <div class="wsp-form-group">
    <label for="exampleFormControlSelect1">Example select</label>
    <select class="wsp-form-control" id="exampleFormControlSelect1">
      <option>1</option>
      <option>2</option>
      <option>3</option>
      <option>4</option>
      <option>5</option>
    </select>
  </div>

  <div class="wsp-form-group">
    <label for="exampleFormControlTextarea1">Example textarea</label>
    <textarea class="wsp-form-control" id="exampleFormControlTextarea1" rows="3"<</textarea>
  </div>
</form>

Sizing

Set heights using classes like .wsp-form-control-lg and .wsp-form-control-sm .

Copy <input class="wsp-form-control wsp-form-control-lg" type="text" placeholder=".wsp-form-control-lg">
<input class="wsp-form-control" type="text" placeholder="Default input">
<input class="wsp-form-control wsp-form-control-sm" type="text" placeholder=".wsp-form-control-sm">
Copy <select class="wsp-form-control wsp-form-control-lg">
  <option>Large select</option>
</select>

<select class="wsp-form-control">
  <option>Default select</option>
</select>

<select class="wsp-form-control wsp-form-control-sm">
  <option>Small select</option>
</select>

Disabled

Add the Disabled boolean attribute on an input to prevent modification of the input’s value.

Copy <input class="wsp-form-control" type="text" value="Readonly input here…" disabled>

Readonly

Add the readonly boolean attribute on an input to prevent modification of the input’s value. Read-only inputs appear lighter (just like disabled inputs), but retain the standard cursor.

Copy <input class="wsp-form-control" type="text" value="Readonly input here…" readonly>

Input Group

Place one add-on or button on either side of an input. You may also place one on both sides of an input.

Copy <div class="wsp-input-group">
  <div class="wsp-input-group-prepend">
    <span class="wsp-input-group-text" id="basic-addon1">
      <i class="wsp-icon wsp-icon-user-circle-o"></i>
    </span>
  </div>
  <input type="text" class="wsp-form-control" placeholder="Username" aria-label="Username" aria-describedby="basic-addon1">
</div>

<div class="wsp-input-group">

    <span class="wsp-input-group-text" id="basic-addon2">
      <i class="wsp-icon wsp-icon-info-circle"></i>

  </div>

you can indicate invalid and valid form fields with .is-invalid and .is-valid.

<input type="text" class="wsp-form-control is-invalid" placeholder="Username" aria-label="Username" >
